What is Privilege Escalation?

Privilege escalation is a security vulnerability that allows an attacker to gain elevated access to resources that are normally protected from an application or user.

Common Types:
  • Vertical Escalation: Gaining higher privileges
  • Horizontal Escalation: Gaining access to other users' resources
  • Root Access: Gaining administrator privileges
  • System Access: Accessing system-level resources

Common Privilege Escalation Vulnerabilities

SUID Files

Files with Set User ID bit that execute with owner's privileges.

Sudo Misconfigurations

Incorrect sudo configurations allowing unauthorized access.

File Permissions

Incorrect file permissions on sensitive system files.

Cron Jobs

Exploitable scheduled tasks running with elevated privileges.
